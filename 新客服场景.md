## 🎯 智能客服记忆功能演示场景

### 场景：电子产品购买咨询

**👤 用户**: 你好，我想买台电脑
**🤖 客服**: 您好！欢迎咨询电子产品。请问您需要笔记本电脑还是台式机？

**👤 用户**: 笔记本
**🤖 客服**: 好的，笔记本有很多品牌。您偏好哪个品牌呢？比如联想、戴尔、苹果等

**👤 用户**: 苹果的
**🤖 客服**: 🎯 根据您的选择（电脑 → 笔记本 → 苹果），为您推荐MacBook系列：
           - MacBook Air 轻薄便携
           - MacBook Pro 性能强劲
    - MacBook Pro with M3芯片 最新款
           您对哪款感兴趣？

**👤 用户**: Pro with M3芯片的
**🤖 客服**: 💡 好的！MacBook Pro with M3芯片 目前有：
           14寸：18,999元 | 16寸：22,999元
           需要了解具体配置吗？

**👤 用户**: 16寸的存储多大？
**🤖 客服**: 📦 16寸MacBook Pro M3 存储选项：
           1TB SSD / 2TB SSD / 4TB SSD
           建议选择2TB版本，性价比最高

**👤 用户**: 好的，那颜色呢？
**🤖 客服**: 🎨 MacBook Pro 16寸颜色：深空灰色、银色
           您偏好哪种？

## 🔧 记忆功能体现

### 1. **产品类型记忆链**

```
电脑 → 笔记本 → 苹果 → MacBook Pro → M3芯片 → 16寸 → 存储 → 颜色
```

### 2. **上下文状态跟踪**

- **阶段1**: 产品大类选择（电脑）
- **阶段2**: 产品细分（笔记本）
- **阶段3**: 品牌确认（苹果）
- **阶段4**: 系列选择（MacBook Pro）
- **阶段5**: 型号细化（M3芯片）
- **阶段6**: 规格确认（16寸）
- **阶段7**: 配置选择（存储）
- **阶段8**: 外观定制（颜色）

### 3. **用户偏好记录**

- 产品类型偏好：笔记本
- 品牌偏好：苹果
- 系列偏好：Pro系列
- 配置倾向：高性能
